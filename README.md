# ScamShield AI - Layer 1 Architecture: Intelligent Extension Node

This component comprises the native client-side web safety environment execution module (**Layer 1 Extension**) developed for the **ScamShield AI** framework under the "Future of Money: Reimagine Finance with AI" theme.

## Dual-Protection Loop Flow
1. **Viewport Detection:** Hooks page initialization loops cleanly at the user's browser runtime layer.
2. **Context Passing:** Forwards target domain signatures asynchronously into Person 1's Spring Boot REST verification endpoint (`/api/check-domain`) hosted on Render via a robust service worker proxy background engine.
3. **DOM Overwrite Mitigation:** Completely replaces standard browser frames with native warning block layouts designed in raw HTML when malicious telemetry signals return true.

## File Map
- `manifest.json`: Configuration mappings, API permission requests, and engine declarations.
- `content.js`: Main interceptor module; reads raw browser viewport paths and alters active documents during incident containment.
- `background.js`: System service worker proxy managing remote network connectivity structures cleanly.
- `popup.html` & `style.css`: Clean, dark-mode administrative system control dashboards.

- **ChatGPT**: Used as an optimization tool for developing clean cross-origin script routines.
